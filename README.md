# N-Ice-Coffee Cafe & Library Sanctuary Portal

A frontend functional Single Page Application (SPA) built for the CSCI390 Web Programming Course (Project Phase 2). This application automates table/workspace reservations, coordinates reading room missing book requests, and deploys a gamified interactive marketing challenge engine.

## 🚀 Key Functional Features
* **Interactive 30s Challenge Arena**: Uses state management loops (`useState`, `useEffect`) to calculate a speed-run countdown timer. Successful runs dynamically unlock a multi-month 15% cafe menu coupon code.
* **Dual Booking Logistics Engine**: Provides controlled structural forms processing client workspace requirements alongside real-time missing literary acquisition requests.
* **Dynamic Content Menu**: Imports strict dataset layers mapping menu specifications dynamically using array loops (`.map()`) backed by explicit element keys.

## 🛠️ Technological Stack Used
* **Frontend Library**: ReactJS (Functional Architectures)
* **Routing Paradigm**: React Router DOM (`BrowserRouter`, `Routes`, `Route`, `Link`)
* **Responsive Styling Framework**: Bootstrap 5 Layout Architecture (Grid Components & Collapse Navbar)

## 📦 Setup and Local Deployment Instructions

1. **Clone the Repository**:
   ```bash
   git clone <your-github-repository-link-here>
   ```

2. **Navigate into the Project Root Directory**:
   ```bash
   cd coffee-shop
   ```

3. **Install Core System Dependencies**:
   ```bash
   npm install
   ```

4. **Boot Up the Local Server Instance**:
   ```bash
   npm start
   ```
   Open your preferred browser window to `http://localhost:3000` to review the portal.
